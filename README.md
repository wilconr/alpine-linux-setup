# Alpine Linux
## Base Installation
* download iso from internet
* create vm, boot
* run `setup-alpine` script
* `reboot`

## Auto Personalization
* login as `root`
* `apk add git`
* `git clone https://github.com/wilconr/alpine-linux-setup`
* `cd alpine-linux-setup`
* `./setup.sh`
